# Claude Cowork : takeaways

*6 juillet 2026*

- « Chat says, Cowork does » : Cowork est un nouveau mode collaboratif. On délègue le travail à Claude, on peut relire/valider ce qu'il a fait.
- Cowork garde la mémoire d'une session à l'autre (contexte, préférences, projets en cours).
- **Claude in Chrome** : fenêtre pour interagir directement avec des apps web (ex. Gmail, recherche de spots de vacances).
- **Plugins** : possibilité de créer des plugins pour des tâches récurrentes, avec un set de skills adapté (ex. skills de data viz).
